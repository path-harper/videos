# AI Media Project

This repository contains AI-generated video content.

## Directory Structure

```
videos/
 ├─ raw/          # Original generated output
 ├─ renders/      # Processed / improved versions
 └─ final/        # Polished version for viewing
```

## Usage

The `videos` directory is the primary storage for video assets, organized by workflow stage:
- `videos/raw/` - Original generated output
- `videos/renders/` - Processed / improved versions  
- `videos/final/` - Polished version for viewing
